# Countdown to New Year
Repo for a javascript clock that countdown to new year! Once it reachs 0 it adds one year again to reset the countdown.

[Visualize](https://geraldopmj.github.io/CountdownNewYear/)

# Technology used:

- Javascript
- HTML
- CSS
- Git e Github
